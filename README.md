# Documentação do Sistema de Planejamento Acadêmico do Curso de CC - Natal

Sistema automatizado para geração de planejamentos acadêmicos otimizados utilizando Teoria dos Grafos (DAGs), desenvolvido para o curso de Ciência da Computação (UERN).

## Tecnologias

| Camada | Tecnologias e Frameworks |
|---|---|
| **Frontend** | React 18, TypeScript, Vite, React Flow |
| **Backend (API)** | Python 3.11, FastAPI, NetworkX, pdfplumber |
| **Infraestrutura** | Docker e Docker Compose |
